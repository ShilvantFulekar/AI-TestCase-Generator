# 🤖 AI Test Case Generator (Agentic AI)

## 🚀 Overview
This project is an AI-powered test case generator that uses OpenAI API to automatically generate structured test cases from feature descriptions.

## 🔥 Features
- Agentic AI architecture (multi-step reasoning)
- Requirement analysis
- Test scenario generation
- Test case creation
- Validation & improvement

## 🛠 Tech Stack
- Python
- OpenAI API
- Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py

Reduces manual effort in QA by automatically generating test cases with better coverage.
